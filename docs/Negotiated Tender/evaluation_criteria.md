# Evaluation Criteria

## Overview

The **Evaluation Criteria** section allows users to define and update the criteria used to evaluate supplier proposals for a Negotiated Tender.

Users can manage criteria directly from the tender **Edit Page** and configure how proposals will be assessed and scored.

---

## Accessing Criteria

### Step 1: Navigate to Edit Page

1. Open the Negotiated Tender record  
2. Navigate to the **Edit Page**  

### Step 2: Click Evaluation Criteria Button

![Edit Page Criteria Button](/img/template/criteria_edit_button.png)

- Click on the **Evaluation Criteria** button
- This opens the criteria list page

---

## Criteria List Page

![Criteria List](/img/template/criteria_list_page.png)

The criteria list page displays:

- All available evaluation criteria  
- Which criteria are currently selected for the tender  
- Status of each criterion
- Options to add or update selected criteria  

---

## Add / Update Criteria

### Step 1: Click Add / Update Button

![Add / Update Button](/img/template/criteria_add_update_button.png)

- Click the **Add / Update** button  
- This will open a modal showing all available criteria options  

### Step 2: Manage Criteria Selection

![Update Criteria Modal](/img/template/criteria_update_modal.png)

Inside the modal:

1. **Select / Remove Criteria** – Use checkboxes to select or unselect evaluation criteria  
   - Only selected criteria will be applied to the tender  
   - Consider proposal quality, cost, capability, and fit
   - Align criteria with negotiation objectives

2. **Review Selections** – Verify that chosen criteria match tender requirements

3. **Save Changes** – Click **Save** to apply the updates  

---

## What Happens After Update

✅ **Immediate Effects:**
- Selected criteria are linked to the tender  
- These criteria will be used to evaluate submitted proposals  
- Changes take effect immediately after saving  
- Suppliers will see evaluation criteria in tender details

ℹ️ **Impact on Process:**
- Proposals will be evaluated using selected criteria
- Evaluation results inform negotiation discussions
- Scores may be revisited during negotiation process
- Final scoring occurs when negotiations conclude
- Transparent criteria support fair supplier treatment

---

## Types of Criteria

Common evaluation criteria for negotiated tenders include:

- **Technical Capability** – Ability to deliver the required services/goods
- **Methodology** – Approach and implementation plan
- **Experience** – Vendor background and proven track record
- **Cost** – Total cost or cost-effectiveness
- **Team Quality** – Capability and experience of key personnel
- **Understanding** – Demonstrated understanding of requirements
- **Compliance** – Adherence to tender specifications

---

## Important Notes

⚠️ **Before Selecting Criteria:**
- Ensure selected criteria align with negotiation objectives
- Consider supplier capabilities and competitive landscape
- Document rationale for criteria selection
- Coordinate with all relevant stakeholders
- Ensure criteria support fair evaluation

⚠️ **After Selecting Criteria:**
- Criteria remain flexible during negotiation phase
- Changes should be documented with clear justification
- Final criteria should be locked before award decision
- Maintain clear audit trail of all modifications
- Only authorized users can update evaluation criteria
- Communicate criteria to all involved stakeholders

---

## Negotiation Phase Considerations

💡 **Note:** Negotiated tenders allow for dynamic evaluation:
- Initial criteria guide proposal assessment
- Criteria may be refined based on supplier responses
- Negotiations may result in criterion adjustments
- Final evaluation should use settled criteria
- Document all criterion changes for transparency
