# Evaluation

## Accessing Evaluation

- Click on **Evaluation**

![Evaluation](/img/settings/management_teams/evaluation/management_teams_evaluation.png)

## Listing Members

![Evaluation Listing](/img/settings/management_teams/evaluation/management_teams_listing.png)

- Displays all members in the evaluation group

## Manage Members

![Manage Evaluation Team](/img/settings/management_teams/evaluation/management_teams_evaluation_listing.png)

Actions available:

1. **Add New Member**
2. **Delete Existing Member**

### Add Team Member

- Click on **Add (1)** button

![Add Evaluation Member](/img/settings/management_teams/evaluation/management_teams_evaluation_add_team.png)

- A modal popup appears
- Select users from the dropdown
- After successful addition, the modal automatically closes

## Notes

- Team structure directly impacts evaluation and approval workflows
- Users can be updated or removed as needed
