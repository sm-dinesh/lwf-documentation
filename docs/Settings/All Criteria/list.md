# Listing of Criterias

![Criteria Listing](/img/settings/criteria/settings_all_criteria_listing.png)

- Displays a list of all available criterias
- Organized based on tender types

## Tabs

### Open Tender (Default)

![Open Tender](/img/settings/criteria/settings_open_tender.png)

- Default active tab
- Displays criterias related to Open Tender

### Other Tabs

- **Negotiated Tender**
- **Simple Procurement**

Users can switch between tabs to create or manage criterias for different tender types.
