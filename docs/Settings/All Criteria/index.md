# All Criterias

Criterias are used to evaluate tender bids based on different parameters. These evaluation points help assess and compare bids effectively during the tendering process.

## Accessing Criterias

![Settings Dashboard](/img/settings/criteria/settings_dashboard.png)

- Navigate to **Settings**
- Click on **All Criterias**

![All Criterias](/img/settings/criteria/settings_all_criteria.png)
