# Bid Invitations

## Overview


Open Tenders are by nature open to all qualified bidders so no direct invitations are required. In Open Tender, qualified bidders/vendors can participate without any prior restrictions.

---

## Accessing Bids

### From Tender List Page

- Click the **Bids** to list all the bids for a tender.

![Create Button](/img/open_tender/open_tender_bids.png)

---

## Bidder Status Tracking

![Create Button](/img/open_tender/open_tender_bids_list.png)

The page list displays important information:

- **Bidder Details** – Name and contact information
- **Submission Status** – Whether bidder has submitted bid
- **Admin Review** – Whether submitted bid has been evaluated
- **Quick Actions** – Evaluation button for rapid assessment

---

## Key Features

✅ **Optional Invitations**
- Open Tenders don't require direct invitations
- Use invitations strategically to build competition
- All bidders can still participate openly

✅ **Status Tracking**
- Monitor bid evaluation in real-time
- Track bid submissions and status
- Quick access to evaluation tools
