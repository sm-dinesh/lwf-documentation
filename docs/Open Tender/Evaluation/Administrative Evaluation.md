## Bid List for Evaluation

![Bid List for Evaluation](/img/open_tender/evaluation/administrative/bid_list_for_evaluation.png)

- Displays all bids available for administrative evaluation.
- Evaluators can select a bid to start evaluation.

---

## Administrative Evaluation Form

![Administrative Evaluation Form](/img/open_tender/evaluation/administrative/administrative_evaluation_form.png)

This form is used to evaluate each bid based on administrative criteria.

### 1. Bidder Name
- Displays the name of the bidder.

### 2. Evaluation Criteria Tab
- Contains all administrative evaluation criteria.

### 3. Documents Tab
- Lists all documents submitted by the bidder.

### 4. Criteria List
- Displays required administrative conditions.

### 5. Yes / No
- Mark each criterion as:
  - **Yes (Passed)**
  - **No (Failed)**

### 6. Rejection Block
- Appears when the bid needs to be rejected.

### 7. Remarks
- Add comments or explanations for evaluation.

---

### Actions

### 8. Submit Evaluation Button
- Submits the evaluation if all criteria are satisfied.

### 9. Reject Bid Button
- Rejects the bid.

---

## Acceptance Rules

- **All the criterion must be marked as "Yes"** to accept a bid.
- **Evaluator Remarks** is required prior before submitting the evaluation.

## Rejection Rules

- At least **one criterion must be marked as "No"** to reject a bid.
- A **rejection reason is mandatory** before submitting rejection.
- **Evaluator Remarks** is required prior before submitting the evaluation.

---

## Documents Tab

![Document Tab](/img/open_tender/evaluation/administrative/administrative_evaluation_document_tab.png)

- Lists all documents related to:
  - The bid  
  - The tender  

- Evaluators can review documents before making decisions.

---

## Confirmation Modals

### Approve Confirmation

![Approve Confirmation](/img/open_tender/evaluation/administrative/approve_confirm_for_administrative_evaluaiton.png)

- Appears when clicking **Submit Evaluation**.
- Form becomes **locked** before final confirmation.

---

### Reject Confirmation

![Reject Confirmation](/img/open_tender/evaluation/administrative/reject_confirm_for_administrative_evaluation.png)

- Appears when clicking **Reject Bid**.
- Requires confirmation before final rejection.

---

## Evaluation Status Tabs

![Pending and Completed Tabs](/img/open_tender/evaluation/administrative/pending_and_completed_evaluation_tab_in_evaluation_page.png)

### 1. Pending Evaluation Tab
- Displays bids that are yet to be evaluated.

### 2. Completed Evaluation Tab
- Displays bids that have already been evaluated.

---

## Completed Evaluations

![Completed Tab](/img/open_tender/evaluation/administrative/completed_tab.png)

- Once a bid is evaluated:
  - It is moved from **Pending** to **Completed**.
  - No further changes can be made.

---

## Summary

This process ensures:

- Structured administrative evaluation of bids  
- Mandatory validation before rejection  
- Secure confirmation with locked forms  
- Clear tracking of pending and completed evaluations  

This guarantees consistency, transparency, and accountability in the evaluation process.
