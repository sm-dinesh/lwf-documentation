## Financial Evaluation Dispute

A **Financial Dispute** occurs when evaluators enter **different financial values or scores** for the same criterion.

In financial evaluation, **all values must match exactly** before the dispute can be resolved.

## Key Rules

- All financial values must be **exactly the same**  
- Dispute is resolved **only after values are aligned**  
- Evaluation **cannot proceed** until resolved  

---

## Summary

Financial Disputes ensure that all financial data is **accurate and consistent across evaluators**.  
Strict matching is required before moving to the next stage.
