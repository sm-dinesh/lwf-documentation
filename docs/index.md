# Welcome

![Logo](https://lwfeprocurements.org/static/images/logo.png)

Welcome to the Lwf eTender Documentation. This platform provides comprehensive guidance for using and managing the Lwf eTender system.

The Lwf eTender is a robust, multi-tenant tender management system developed by **Studio Matrix (Nepal)** for **The Lutheran World Federation (LWF)**. It streamlines the entire procurement process, from tender creation and submission to evaluation and award, ensuring transparency, efficiency, and ease of use across multiple organizations.

This documentation covers everything you need to get started, including user guides, administrative workflows, and system features, making it your go-to reference for operating and understanding Lwf eTender.
