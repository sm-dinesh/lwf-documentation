# Welcome to Lwf eTender Documentation

![Logo](https://lwfeprocurements.org/static/images/logo.png)

Welcome to the comprehensive Lwf eTender Documentation. This platform provides complete guidance for using and managing the Lwf eTender system.

---

## 📋 About Lwf eTender

The **Lwf eTender** is a robust, multi-tenant tender management system developed by **Studio Matrix (Nepal)** for **The Lutheran World Federation (LWF)**. 

It streamlines the entire procurement process, from tender creation and submission to evaluation and award, ensuring:

- ✅ **Transparency** – Clear, visible processes for all stakeholders
- ✅ **Efficiency** – Streamlined workflows reducing manual effort
- ✅ **Accountability** – Complete audit trails and documentation
- ✅ **Fairness** – Equitable treatment of all suppliers

---

## 🎯 Documentation Scope

This documentation covers everything you need to:

- **Create & Manage Tenders** – Set up procurement processes
- **Configure Evaluation** – Define selection criteria
- **Manage Suppliers** – Invite and track vendor participation
- **Process Bids** – Receive, evaluate, and award contracts
- **Ensure Compliance** – Maintain audit trails and documentation

## 📚 Procurement Modules

Lwf eTender supports three main procurement types:

### 1. 🏪 [Simple Procurement](./Simple%20Procurement/index.md)
**For low-value procurements (€200 – €9,999)**
- Quick Request for Quotation (RFQ) process
- Pre-identified vendors
- Streamlined evaluation
- Fast-track approval

### 2. 🏢 [Open Tender](./Open%20Tender/index.md)
**For high-value procurements (€10,000+)**
- Open and competitive bidding
- Multiple suppliers welcome
- Comprehensive evaluation
- Full transparency and compliance

### 3. 🤝 [Negotiated Tender](./Negotiated%20Tender/index.md)
**For specialized procurements**
- Direct supplier negotiation
- Limited participant selection
- Flexible terms discussion
- Suitable for complex services

## ⚡ Important Concepts

### Procurement Value Ranges

| Procurement Type | Value Range | Use Case |
|---|---|---|
| Simple Procurement | €200 – €9,999 | Quick RFQ process with known vendors |
| Open Tender | €10,000+ | Competitive bidding with new suppliers |
| Negotiated Tender | Any | Specialized services or complex negotiations |

### Tender Lifecycle States

1. **Draft** – Tender is being prepared (not visible to suppliers)
2. **Published** – Tender is active (suppliers can submit bids)
3. **Closed** – Submission deadline has passed (no more submissions accepted)
4. **Evaluated** – Bids have been assessed and scored
5. **Awarded** – Winner selected and contract issued

## 📝 Best Practices

✅ **Planning Phase**
- Define clear procurement objectives
- Set realistic timelines
- Identify eligible suppliers early

✅ **Configuration Phase**
- Use clear, specific evaluation criteria
- Test templates before using
- Document all settings for audit trail

✅ **Execution Phase**
- Monitor submission deadlines
- Keep detailed records of all actions
- Communicate transparently with all suppliers

✅ **Closure Phase**
- Document evaluation results
- Maintain complete audit trail
- Archive all tender documentation
