### Navigating the Listing Page

The Listing Page allows you to track existing demaind notes and initiate new ones.

-   **Search/Filter:** Use the filter bar to find previous Demand Notes by Status, Date, or Created by.
    
-   **Initiating New Entry:** Click the **[+ Create]** button located at the top right of the page.
