### Creating the Demand Note

-   **Visual:** 


![Create Demand Note](/img/demand_note/demand_note_create.png)

The Demand Note form consists of 8 fields under 3 sections:

#### i. Budget Section
1.  **Select Budget**  – Select the associated buget from the available options.


#### ii. Basic Information
1.  **Comment**  – Enter comment for the demand note.
2.  **Reference**  – Enter a unique reference code that represents the demand note.
3.  **Select Currency**  – Select the currency that is valid for the demand note.
4.  **Location**  – Enter a valid location.
5.  **Expected Date**  – Enter a future expected date.

#### iii. Procurement Items
In this section, you need to enter the details of the items that are required in demand note which includes Item and Quantity. A demand note can contains only one item or multiple items as well.

**Save Section**

Save and publish the demand note.
