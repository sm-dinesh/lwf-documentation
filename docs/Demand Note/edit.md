# Edit Demand Note

The **Edit Demand Note** page allows users to modify an existing demand note. Users can update previously entered information, including budget details and other sections created during the initial setup.

## Accessing the Edit Page

- Navigate to the Demand Note listing page
- Select a Demand Note
- Click on the **Edit** button from the details page

## Overview

![Edit Demand Note](/img/demand_note/demand_note_edit.png)

This page provides editable fields for updating the demand note. All previously entered data is pre-filled for convenience.

## Editable Sections

### Budget Details
- Users can view the budget-related information associated with the demand note.

### Other Information
- All fields entered during the creation of the demand note can be modified as needed.

## Saving Changes

- After making changes, click on the **Save Section** button.
- This will update the corresponding section of the demand note.

## Available Actions

At the top of the page, the following action buttons are available:

### Go Back
- Navigates the user to the previous page.

### View
- Redirects to the Demand Note Details page.
- Allows users to review the full demand note in read-only mode.

### Send for Review
- Submits the demand note for the approval workflow.
- The note will go through different authorization stages such as:
  - Reviewer
  - Checker
  - Approver

### Add File
- Allows users to upload and attach additional files to the demand note.

## Notes

- Ensure all required fields are updated before saving.
- Once sent for review, further edits may be restricted depending on workflow permissions.
