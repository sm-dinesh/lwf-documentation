# Add File to Demand Note

The **Add File** feature allows users to upload and attach supporting documents to a Demand Note. This helps in maintaining all relevant files in one place for reference and approval processes.

## Accessing Add File

- Navigate to the Demand Note Edit page
- Click on the **Add File** button at the top of the page

## Overview

![Add File](/img/demand_note/demand_note_add_file.png)

A modal popup will appear, allowing users to upload files.

## Uploading Files

- Click on the file uploader area
- Select a file from your local file browser
- Upload the selected file

## Features

- Supports uploading files directly from your system
- Simple and quick file attachment process
- Files are linked to the respective Demand Note

## Notes

- Ensure the file size and format meet system requirements
- Uploaded files will be visible under the **Files** tab in the Demand Note Details page
