# Simple Procurement Evaluation

## Overview

The **Evaluation** page allows procurement managers to review, score, and approve or reject vendor bids.  
Evaluations are performed based on predefined **Evaluation Criteria** and financial documents submitted by vendors.

---

## Evaluation Form

![Evaluation Form](/img/simple_procurement/evaluations/simple_procurement_evaluation_form.png)

Key fields and sections:

1. **Bidder Name & Email**  
   - Displays vendor information with a link to their profile  

2. **Financial Documents**  
   - Documents submitted by the vendor during bid application  

3. **Currency Selection**  
   - Choose the currency for evaluating financial bids  

4. **Items List**  
   - Items from the related demand note  
   - Includes unit price fields for evaluation  

5, 6, 7. **Evaluation Criteria**  
   - Displays criteria selected during procurement creation  
   - Fill in scores or values as per each criterion  

8. **Save Evaluation Button**  
   - Click to save the evaluation results  

> **Special Note:**  
> VAT criteria is automatically calculated based on selected values. There is **no manual input** for VAT in the evaluation form.

---

## Rejection Section

![Rejection on Evaluation](/img/simple_procurement/evaluations/rejection_on_evaluation.png)

1. **Rejection Blocks Heading**  
   - Marks the start of rejection options  

2. **Rejection Checklist**  
   - Predefined reasons for rejecting a bid  

3. **Other Rejection Reason**  
   - Text area for custom rejection reasons  

4. **Save Evaluation Button**  
   - Click to finalize evaluation  

**Important Rules:**

- If any rejection reason is filled, the bid is **rejected**  
- If no rejection reason is selected or entered, the bid is **approved**
