# Evaluation

## Overview

The **Evaluation** phase in Simple Procurement involves reviewing and comparing vendor quotations submitted in response to the Request for Quotation (RFQ). This process ensures fair, transparent, and objective selection of the most competitive offer based on predefined evaluation criteria.

During evaluation, quotations are assessed using a **Comparative Evaluation template**, which structures the comparison and scoring. The system facilitates side-by-side analysis, automated calculations, and documentation of the decision-making process.

---

## 🎯 Purpose

The evaluation process aims to:

- Compare quotations systematically using standardized criteria  
- Calculate scores and rankings automatically  
- Identify the most competitive offer  
- Maintain an audit trail of the evaluation decisions  
- Prepare results for approval by the Core Management Team  

---

## ⚙️ Evaluation Workflow

### 1. 📥 Access Evaluation Page

1. Navigate to the Simple Procurement record  
2. Ensure quotations have been submitted by vendors  
3. Click on the **Evaluation** tab or button  

### 2. 📊 Review Submitted Quotations

- View all received quotations in a list or grid format  
- Access detailed quotation documents and attachments  
- Note any clarifications or amendments if needed  

### 3. 🔍 Apply Evaluation Criteria

- Open the **Comparative Evaluation template**  
- The template includes predefined criteria (e.g., price, quality, delivery time)  
- Assign weights and scores to each criterion for each quotation  

### 4. 📈 Calculate and Compare

- The system automatically calculates total scores based on weights  
- View comparative charts, tables, or dashboards  
- Identify the highest-scoring quotation  

### 5. ✅ Finalize Selection

- Select the recommended vendor and quotation  
- Add comments or justifications for the decision  
- Submit the evaluation results for approval  

---

## 📋 Key Features

- **Automated Scoring**: Reduces manual errors and ensures consistency  
- **Weighted Criteria**: Allows customization of importance for different factors  
- **Audit Trail**: Logs all changes and decisions for transparency  
- **Integration**: Seamlessly connects to approval workflows
