# Edit Simple Procurement


## Overview

The **Edit** page allows users to update details of an existing Simple Procurement.  
This is typically used when corrections or updates are required before or after publishing (depending on permissions).

---

![Edit Simple Procurement](/img/simple_procurement/simple_procurement_edit.png)

## Fields

The following fields are available for editing as shown in the image:

1. **Name Field**  
   Update the procurement title to better reflect the purpose or scope.

2. **Category**  
   Select or modify the procurement category (e.g., Goods, Services, Works).

3. **Terms and Conditions**  
   Define or update the rules, requirements, and obligations for suppliers participating in the procurement.

4. **Published Date**  
   Set or adjust the date when the procurement becomes officially published.

5. **Submission Date**  
   Specify the deadline for suppliers to submit their quotations.

6. **Opening Date**  
   Define the date when submitted quotations will be opened and reviewed.

7. **Code**  
   Update the unique procurement code used for identification and tracking.

8. **Update Note** *(Required)*  
   Provide a reason or description for the changes made.  
   This ensures proper tracking and audit history of updates.

---

## How to Edit

1. Open the Simple Procurement record  
2. Click on the **Edit** option  
3. Update the required fields  
4. Enter a valid **Update Note**  
5. Save the changes  

---

## Important Notes

- The **Update Note is mandatory** for every edit  
- Some fields may be restricted after publishing  
- Ensure all dates are accurate to avoid process issues  
- Changes may be logged for audit and tracking purposes
