# Evaluation Criteria

## Overview

The **Evaluation Criteria** section allows users to define and update the criteria used to evaluate supplier quotations for a Simple Procurement.

Users can manage criteria directly from the procurement **Edit Page** and configure how quotations will be assessed and scored.

---

## Accessing Criteria

### Step 1: Navigate to Edit Page

1. Open the Simple Procurement record  
2. Navigate to the **Edit Page**  

### Step 2: Click Evaluation Criteria Button

![Edit Page Criteria Button](/img/template/criteria_edit_button.png)

- Click on the **Evaluation Criteria** button
- This opens the criteria list page

---

## Criteria List Page

![Criteria List](/img/template/criteria_list_page.png)

The criteria list page displays:

- All available evaluation criteria  
- Which criteria are currently selected for the procurement  
- Status of each criterion
- Options to add or update selected criteria  

---

## Add / Update Criteria

### Step 1: Click Add / Update Button

![Add / Update Button](/img/template/criteria_add_update_button.png)

- Click the **Add / Update** button  
- This will open a modal showing all available criteria options  

### Step 2: Manage Criteria Selection

![Update Criteria Modal](/img/template/criteria_update_modal.png)

Inside the modal:

1. **Select / Remove Criteria** – Use checkboxes to select or unselect evaluation criteria  
   - Only selected criteria will be applied to the procurement  
   - Consider all relevant aspects (price, quality, delivery, capability, etc.)

2. **Review Selections** – Verify that chosen criteria match procurement requirements

3. **Save Changes** – Click **Save** to apply the updates  

---

## What Happens After Update

✅ **Immediate Effects:**
- Selected criteria are linked to the procurement  
- These criteria will be used during the evaluation process  
- Changes take effect immediately after saving  
- Vendors will see evaluation criteria in procurement details

ℹ️ **Impact on Process:**
- Quotations will be evaluated using selected criteria
- Evaluation scores will be calculated based on these criteria
- Comparison will focus on selected metrics only
- Clear criteria ensure fair and consistent evaluation

---

## Types of Criteria

Common evaluation criteria include:

- **Price/Cost** – Total cost or unit pricing
- **Quality** – Product or service quality standards
- **Delivery** – Timeline and availability
- **Capability** – Vendor capacity and experience
- **Compliance** – Adherence to requirements
- **References** – Track record and performance history

---

## Important Notes

⚠️ **Before Selecting Criteria:**
- Ensure selected criteria match the procurement requirements
- Align criteria with procurement scope and budget
- Consider vendor capabilities in selecting appropriate criteria
- Document criteria selection rationale

⚠️ **After Selecting Criteria:**
- Removing criteria may affect ongoing evaluations
- Significant changes may require stakeholder review
- Changes are logged for audit purposes
- Only authorized users can update evaluation criteria
- Communicate criteria changes to relevant stakeholders
