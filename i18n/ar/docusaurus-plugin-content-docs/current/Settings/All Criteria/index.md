# جميع المعايير

تُستخدم المعايير لتقييم عطاءات المناقصات بناءً على معايير مختلفة. تساعد نقاط التقييم هذه في تقييم ومقارنة العطاءات بشكل فعال خلال عملية المناقصة.

## الوصول إلى المعايير

![لوحة إعدادات القيادة](/img/settings/criteria/settings_dashboard.png)

- انتقل إلى **الإعدادات**
- انقر على **جميع المعايير**

![جميع المعايير](/img/settings/criteria/settings_all_criteria.png)
