# قائمة المعايير

![قائمة المعايير](/img/settings/criteria/settings_all_criteria_listing.png)

- يعرض قائمة بجميع المعايير المتاحة
- منظم بناءً على أنواع المناقصات

## علامات التبويب

### المناقصة المفتوحة (افتراضي)

![المناقصة المفتوحة](/img/settings/criteria/settings_open_tender.png)

- علامة التبويب النشطة افتراضيًا
- يعرض المعايير المتعلقة بالمناقصة المفتوحة

### علامات التبويب الأخرى

- **المناقصة المتفاوض عليها**
- **المشتريات البسيطة**

يمكن للمستخدمين التبديل بين علامات التبويب لإنشاء أو إدارة المعايير لأنواع المناقصات المختلفة.
