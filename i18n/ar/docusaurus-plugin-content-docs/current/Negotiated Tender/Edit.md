# تحرير المناقصة المتفاوض عليها

## نظرة عامة

تتيح صفحة **التحرير** للمستخدمين تحديث تفاصيل مناقصة متفاوض عليها موجودة.  
يُستخدم هذا عادةً عندما تكون هناك حاجة لتصحيحات أو تحديثات قبل النشر أو بعده (حسب الأذونات).

---

## حقول التحرير

يمكن تحرير الحقول التالية:

1. **الاسم**  
   تحديث عنوان المناقصة ليعكس الغرض أو النطاق بشكل أفضل

2. **الفئة**  
   تحديد أو تعديل فئة المشتريات (مثل السلع، الخدمات، الأشغال)

3. **الوصف**  
   تحديث الوصف التفصيلي لنطاق المشتريات ومتطلباتها

4. **الشروط والأحكام**  
   تحديد أو مراجعة القواعد والمتطلبات والالتزامات للموردين

5. **تاريخ النشر**  
   تحديد أو ضبط التاريخ الذي تنشر فيه المناقصة رسمياً

6. **تاريخ التقديم**  
   تحديد الموعد النهائي للموردين لتقديم عروضهم

7. **تاريخ الفتح**  
   تحديد التاريخ الذي سيتم فيه فتح العروض المقدمة ومراجعتها

8. **الرمز**  
   تحديث رمز المناقصة الفريد (إذا كان مسموحاً به قبل النشر)

9. **ملاحظة التحديث** *(مطلوب)*  
   تقديم سبب أو وصف للتغييرات التي تم إجراؤها  
   يضمن ذلك التتبع السليم وسجل التدقيق للتحديثات

---

## كيفية التحرير

1. افتح سجل المناقصة المتفاوض عليها
2. انقر فوق خيار **تحرير**
3. قم بتحديث الحقول المطلوبة
4. أدخل **ملاحظة تحديث** مفصلة تشرح التغييرات
5. احفظ التغييرات

## ملاحظات مهمة

- **ملاحظة التحديث إلزامية** لكل عملية تحرير
- قد تكون بعض الحقول مقيدة بعد النشر
- التأكد من دقة جميع التواريخ وأنها في تسلسل منطقي
- قد يتم تسجيل التغييرات لأغراض التدقيق والتتبع
- مراجعة التغييرات بعناية قبل الحفظ
