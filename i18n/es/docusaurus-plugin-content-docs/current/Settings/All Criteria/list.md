# Lista de criterios

![Lista de criterios](/img/settings/criteria/settings_all_criteria_listing.png)

- Muestra una lista de todos los criterios disponibles
- Organizados según los tipos de licitación

## Pestañas

### Licitación abierta (predeterminada)

![Licitación abierta](/img/settings/criteria/settings_open_tender.png)

- Pestaña activa por defecto
- Muestra los criterios relacionados con la licitación abierta

### Otras pestañas

- **Licitación negociada**
- **Contratación simple**

Los usuarios pueden cambiar entre pestañas para crear o gestionar criterios para diferentes tipos de licitación.
