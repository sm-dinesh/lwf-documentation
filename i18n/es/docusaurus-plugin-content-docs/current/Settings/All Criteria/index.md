# Todos los criterios

Los criterios se utilizan para evaluar las ofertas de licitación en función de diferentes parámetros. Estos puntos de evaluación ayudan a evaluar y comparar las ofertas de manera efectiva durante el proceso de licitación.

## Acceso a los criterios

![Panel de configuración](/img/settings/criteria/settings_dashboard.png)

- Navegar a **Configuración**
- Hacer clic en **Todos los criterios**

![Todos los criterios](/img/settings/criteria/settings_all_criteria.png)
