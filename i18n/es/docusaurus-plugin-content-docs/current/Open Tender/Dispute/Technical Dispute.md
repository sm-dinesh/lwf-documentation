## Reglas clave

- Las puntuaciones **no tienen que coincidir** para resolver una disputa técnica  
- El equipo de adquisiciones revisa y decide **Aceptar** o **Rechazar**  
- La edición de puntuaciones es opcional; el sistema permite la aprobación final incluso con diferencias  

---

## Resumen

Las disputas técnicas garantizan que **todas las evaluaciones técnicas se revisen de manera justa**.  
Los usuarios de adquisiciones tienen **flexibilidad para resolver disputas sin requerir una coincidencia exacta de puntos**.
