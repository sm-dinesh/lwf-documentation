# Enviar Nota de demanda para revisión

La función **Enviar para revisión** permite a los usuarios enviar una Nota de demanda al flujo de trabajo de aprobación. La nota de demanda será revisada y aprobada por usuarios designados en diferentes roles.

## Acceder a Enviar para revisión

- Navegue a la página de edición de la Nota de demanda
- Haga clic en el botón **Enviar para revisión** en la parte superior de la página

## Descripción general

![Enviar para revisión](/img/demand_note/demand_note_send_for_review.png)

Aparecerá una ventana emergente donde los usuarios pueden seleccionar aprobadores de diferentes roles.

## Roles de aprobación

El modal contiene tres niveles de aprobación, indicados en la imagen:

### 1. Revisor
- Primer nivel de revisión
- Responsable de la verificación inicial de la nota de demanda

### 2. Verificador
- Segundo nivel de validación
- Garantiza la exactitud e integridad de los datos

### 3. Aprobador
- Autoridad final para la aprobación
- Confirma y aprueba la nota de demanda para su posterior procesamiento

## Envío de invitaciones

- Seleccione usuarios para cada rol (Revisor, Verificador, Aprobador)
- Una vez seleccionado, envíe el formulario
- Se enviarán invitaciones a los respectivos usuarios para su aprobación

## Notas

- Asegúrese de que se seleccionen los usuarios correctos para cada rol antes de enviar
- El proceso de aprobación sigue la jerarquía definida: Revisor → Verificador → Aprobador
- Después del envío, los permisos de edición pueden estar restringidos según la configuración del flujo de trabajo
