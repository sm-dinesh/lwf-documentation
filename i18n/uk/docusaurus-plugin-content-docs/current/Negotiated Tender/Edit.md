# Редагування переговорної процедури

## Огляд

Сторінка **Редагування** дозволяє користувачам оновлювати деталі існуючої переговорної процедури.  
Зазвичай це використовується, коли потрібні виправлення або оновлення до або після публікації (залежно від прав доступу).

---

## Поля для редагування

Наступні поля можна редагувати:

1. **Назва**  
   Оновіть назву тендеру, щоб вона краще відображала мету або обсяг

2. **Категорія**  
   Виберіть або змініть категорію закупівлі (наприклад, Товари, Послуги, Роботи)

3. **Опис**  
   Оновіть детальний опис обсягу закупівлі та вимог

4. **Правила та умови**  
   Визначте або перегляньте правила, вимоги та зобов'язання для постачальників

5. **Дата публікації**  
   Встановіть або налаштуйте дату, коли тендер буде офіційно опублікований

6. **Дата подання**  
   Вкажіть кінцевий термін подання постачальниками своїх пропозицій

7. **Дата відкриття**  
   Визначте дату, коли подані пропозиції будуть відкриті та переглянуті

8. **Код**  
   Оновіть унікальний код тендеру (якщо це дозволено до публікації)

9. **Примітка про оновлення** *(Обов'язково)*  
   Надайте причину або опис внесених змін  
   Це забезпечує належне відстеження та історію аудиту оновлень

---

## Як редагувати

1. Відкрийте запис про переговорну процедуру
2. Натисніть на опцію **Редагувати**
3. Оновіть необхідні поля
4. Введіть детальну **Примітку про оновлення** з поясненням змін
5. Збережіть зміни

## Важливі примітки

- **Примітка про оновлення є обов'язковою** для кожного редагування
- Деякі поля можуть бути обмежені для редагування після публікації
- Переконайтеся, що всі дати точні та вказані в логічній послідовності
- Зміни можуть фіксуватися для аудиту та відстеження
- Уважно перегляньте зміни перед збереженням
