# Критерії переговорного тендеру

Процес подібний до відкритого тендеру.

## Адміністративні критерії

- Ті самі поля та правила, що й в адміністративній частині відкритого тендеру

![Адміністративні критерії](/img/settings/criteria/settings_open_tender_administrative_create.png)

## Технічні критерії

![Технічні критерії](/img/settings/criteria/settings_open_tender_technical_create.png)

## Фінансові критерії

- Ті самі поля та правила, що й у фінансовій частині відкритого тендеру

![Фінансові критерії](/img/settings/criteria/settings_open_tender_financial_create.png)

### Дії

- **Скасувати** – Відхилити зміни
- **Створити критерій** – Зберегти критерій

## Примітки

- Переконайтеся, що розподіл ваги є правильним при використанні ієрархічних критеріїв
- Правила валідації повинні бути чітко визначені, щоб уникнути помилок під час оцінки
- Структура критеріїв безпосередньо впливає на результати оцінки пропозицій
