# Усі критерії

Критерії використовуються для оцінки тендерних пропозицій на основі різних параметрів. Ці пункти оцінки допомагають ефективно оцінювати та порівнювати пропозиції під час тендерного процесу.

## Доступ до критеріїв

![Панель налаштувань](/img/settings/criteria/settings_dashboard.png)

- Перейдіть до **Налаштування**
- Натисніть на **Усі критерії**

![Усі критерії](/img/settings/criteria/settings_all_criteria.png)
