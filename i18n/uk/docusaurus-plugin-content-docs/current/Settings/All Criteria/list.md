# Список критеріїв

![Список критеріїв](/img/settings/criteria/settings_all_criteria_listing.png)

- Відображає список усіх доступних критеріїв
- Організовано за типами тендерів

## Вкладки

### Відкритий тендер (за замовчуванням)

![Відкритий тендер](/img/settings/criteria/settings_open_tender.png)

- Активна вкладка за замовчуванням
- Відображає критерії, пов'язані з відкритим тендером

### Інші вкладки

- **Переговорний тендер**
- **Спрощена закупівля**

Користувачі можуть перемикатися між вкладками для створення або керування критеріями для різних типів тендерів.
