# Оцінка спрощених закупівель

## Огляд

Сторінка **Оцінка** дозволяє менеджерам із закупівель переглядати, оцінювати та затверджувати або відхиляти пропозиції постачальників.  
Оцінка проводиться на основі заздалегідь визначених **Критеріїв оцінки** та фінансових документів, поданих постачальниками.

---

## Форма оцінки

![Форма оцінки](/img/simple_procurement/evaluations/simple_procurement_evaluation_form.png)

Основні поля та розділи:

1. **Ім'я та Email учасника**  
   - Відображає інформацію про постачальника з посиланням на його профіль  

2. **Фінансові документи**  
   - Документи, подані постачальником під час подання пропозиції  

3. **Вибір валюти**  
   - Виберіть валюту для оцінки фінансових пропозицій  

4. **Список товарів**  
   - Товари з відповідної заявки на потребу  
   - Містить поля ціни за одиницю для оцінки  

5, 6, 7. **Критерії оцінки**  
   - Відображає критерії, вибрані під час створення закупівлі  
   - Заповніть бали або значення відповідно до кожного критерію  

8. **Кнопка «Зберегти оцінку»**  
   - Натисніть, щоб зберегти результати оцінки  

> **Спеціальна примітка:**  
> Критерій ПДВ розраховується автоматично на основі вибраних значень. У формі оцінки **немає ручного введення** ПДВ.

---

## Розділ відхилення

![Відхилення при оцінці](/img/simple_procurement/evaluations/rejection_on_evaluation.png)

1. **Заголовок блоків відхилення**  
   - Позначає початок параметрів відхилення  

2. **Контрольний список відхилення**  
   - Заздалегідь визначені причини для відхилення пропозиції  

3. **Інша причина відхилення**  
   - Текстова область для власних причин відхилення  

4. **Кнопка «Зберегти оцінку»**  
   - Натисніть, щоб завершити оцінку  

**Важливі правила:**

- Якщо заповнена будь-яка причина відхилення, пропозиція вважається **відхиленою**  
- Якщо причину відхилення не вибрано і не введено, пропозиція вважається **затвердженою**
