# Надіслати Заявку на закупівлю на розгляд

Функція **Надіслати на розгляд** дозволяє користувачам подавати Заявку на закупівлю в процес затвердження. Заявка буде розглянута та затверджена призначеними користувачами з різними ролями.

## Доступ до надсилання на розгляд

- Перейдіть на сторінку редагування Заявки на закупівлю
- Натисніть кнопку **Надіслати на розгляд** у верхній частині сторінки

## Огляд

![Надіслати на розгляд](/img/demand_note/demand_note_send_for_review.png)

З'явиться модальне вікно, де користувачі можуть вибрати затверджувачів з різними ролями.

## Ролі затвердження

Модальне вікно містить три рівні затвердження, як показано на зображенні:

### 1. Рецензент (Reviewer)
- Перший рівень перевірки
- Відповідає за первинну верифікацію заявки на закупівлю

### 2. Перевіряючий (Checker)
- Другий рівень валідації
- Забезпечує правильність та повноту даних

### 3. Затверджувач (Approver)
- Остаточна інстанція для затвердження
- Підтверджує та затверджує заявку на закупівлю для подальшої обробки

## Надсилання запрошень

- Виберіть користувачів для кожної ролі (Рецензент, Перевіряючий, Затверджувач)
- Після вибору надішліть форму
- Запрошення будуть надіслані відповідним користувачам для затвердження

## Примітки

- Перед надсиланням переконайтеся, що для кожної ролі вибрано правильних користувачів
- Процес затвердження відбувається згідно з визначеною ієрархією: Рецензент → Перевіряючий → Затверджувач
- Після подання права на редагування можуть бути обмежені залежно від налаштувань робочого процесу
