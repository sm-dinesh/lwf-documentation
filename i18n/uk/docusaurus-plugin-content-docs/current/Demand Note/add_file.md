# Додати файл до Заявки на закупівлю

Функція **Додати файл** дозволяє користувачам завантажувати та додавати супровідні документи до Заявки на закупівлю. Це допомагає зберігати всі відповідні файли в одному місці для довідки та процесів затвердження.

## Доступ до додавання файлу

- Перейдіть на сторінку редагування Заявки на закупівлю
- Натисніть кнопку **Додати файл** у верхній частині сторінки

## Огляд

![Додати файл](/img/demand_note/demand_note_add_file.png)

З'явиться модальне вікно, що дозволить користувачам завантажувати файли.

## Завантаження файлів

- Натисніть на область завантаження файлів
- Виберіть файл із локального браузера файлів
- Завантажте вибраний файл

## Особливості

- Підтримує завантаження файлів безпосередньо з вашої системи
- Простий і швидкий процес додавання файлів
- Файли пов'язані з відповідною Заявкою на закупівлю

## Примітки

- Переконайтеся, що розмір і формат файлу відповідають системним вимогам
- Завантажені файли будуть відображатися на вкладці **Файли** на сторінці деталей Заявки на закупівлю
