# Редагувати Заявку на закупівлю

Сторінка **Редагувати Заявку на закупівлю** дозволяє користувачам змінювати існуючу заявку. Користувачі можуть оновлювати раніше введену інформацію, включаючи деталі бюджету та інші розділи, створені під час початкового налаштування.

## Доступ до сторінки редагування

- Перейдіть на сторінку списку Заявок на закупівлю
- Виберіть Заявку на закупівлю
- Натисніть кнопку **Редагувати** на сторінці деталей

## Огляд

![Редагувати Заявку на закупівлю](/img/demand_note/demand_note_edit.png)

Ця сторінка містить поля для редагування для оновлення заявки на закупівлю. Усі раніше введені дані заповнюються автоматично для зручності.

## Розділи для редагування

### Деталі бюджету
- Користувачі можуть переглядати інформацію про бюджет, пов'язану із заявкою на закупівлю.

### Інша інформація
- Усі поля, заповнені під час створення заявки на закупівлю, можна змінити за потреби.

## Збереження змін

- Після внесення змін натисніть кнопку **Зберегти розділ**.
- Це оновить відповідний розділ заявки на закупівлю.

## Доступні дії

У верхній частині сторінки доступні наступні кнопки дій:

### Назад
- Повертає користувача на попередню сторінку.

### Перегляд
- Перенаправляє на сторінку деталей Заявки на закупівлю.
- Дозволяє користувачам переглядати повну заявку в режимі лише для читання.

### Надіслати на розгляд
- Подає заявку на закупівлю для процесу затвердження.
- Заявка пройде через різні етапи авторизації, такі як:
  - Рецензент (Reviewer)
  - Перевіряючий (Checker)
  - Затверджувач (Approver)

### Додати файл
- Дозволяє користувачам завантажувати та додавати додаткові файли до заявки на закупівлю.

## Примітки

- Перед збереженням переконайтеся, що всі обов'язкові поля оновлені.
- Після надсилання на розгляд подальше редагування може бути обмежене залежно від дозволів робочого процесу.
