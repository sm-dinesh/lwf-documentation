## Différend d'évaluation financière

Un **différend financier** survient lorsque les évaluateurs saisissent des **valeurs financières ou des scores différents** pour le même critère.

Dans l'évaluation financière, **toutes les valeurs doivent correspondre exactement** avant que le différend ne puisse être résolu.

4. Bouton Rejeter (confirmer)  

---

## Règles clés

- Toutes les valeurs financières doivent être **exactement les mêmes**  
- Le différend est résolu **uniquement après l'alignement des valeurs**  
- L'évaluation **ne peut pas progresser** tant qu'elle n'est pas résolue  

---

## Résumé

Les différends financiers garantissent que toutes les données financières sont **exactes et cohérentes entre les évaluateurs**.  
Une correspondance stricte est requise avant de passer à l'étape suivante.
