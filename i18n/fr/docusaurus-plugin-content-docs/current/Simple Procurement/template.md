# Gestion des modèles

## Aperçu

La fonction **Gestion des modèles** permet aux utilisateurs de gérer, modifier et télécharger des modèles de documents associés à une procédure simplifiée.  

Les utilisateurs peuvent :
- Ajouter de nouveaux modèles à un approvisionnement
- Modifier les détails des modèles existants
- Visualiser les espaces réservés (placeholders) des modèles
- Générer des documents avec ou sans remplacement des espaces réservés
- Télécharger des documents dans plusieurs formats

---

## Accéder à la gestion des modèles

1. Ouvrez la **Page de modification** de la procédure simplifiée  
2. Cliquez sur le **Bouton Modèle** pour accéder à la liste des modèles

![Bouton Modèle Page de modification](/img/template/template_edit_button.png)

---

## Page de la liste des modèles

![Page de la liste des modèles](/img/template/template_list_page.png)

La page de la liste des modèles affiche :

- Tous les modèles liés à l'approvisionnement  
- Les noms et descriptions des modèles
- Les actions disponibles pour chaque modèle
- L'option pour ajouter de nouveaux modèles

---

## Ajouter un modèle

### Étape 1 : Cliquer sur le bouton Ajouter un modèle

![Bouton Ajouter un modèle](/img/template/template_add_button.png)

- Cliquez sur le bouton **Ajouter un modèle** pour ouvrir la fenêtre modale de sélection

### Étape 2 : Sélectionner un modèle et ajouter

![Fenêtre modale d'ajout de modèle](/img/template/template_add_modal.png)

À l'intérieur de la fenêtre modale :

1. **Liste déroulante de sélection** – Choisissez un modèle dans la liste disponible  
2. **Bouton Ajouter** – Cliquez pour lier le modèle sélectionné à l'approvisionnement  

---

## Actions sur les modèles

![Boutons d'action sur les modèles](/img/template/template_action_buttons.png)

Pour chaque modèle, les actions suivantes sont disponibles :

1. **Bouton d'action Modifier** – Ouvre les options de modification du modèle  
2. **Bouton Modifier** – Modifier les détails du modèle  
3. **Bouton Espace réservé** – Visualiser les espaces réservés utilisés dans le modèle  
4. **Télécharger le modèle** – Télécharger le modèle brut sans remplacement des espaces réservés  
5. **Télécharger Word** – Télécharger le document Word avec les espaces réservés remplacés  
6. **Télécharger PDF** – Télécharger le document PDF avec les espaces réservés remplacés  

---

## Modifier un modèle

### Étape 1 : Cliquer sur l'action Modifier

![Fenêtre modale de modification de modèle](/img/template/template_edit_modal.png)

À l'intérieur de la fenêtre modale de modification :

1. **Modification du nom du modèle** – Modifier le titre du modèle  
2. **Bouton Ouvrir l'éditeur** – Lancer la fenêtre de l'éditeur de modèle  

### Étape 2 : Utiliser l'éditeur de modèle

![Fenêtre de l'éditeur de modèle](/img/template/template_editor_window.png)

- Une nouvelle fenêtre contextuelle s'ouvre pour modifier le contenu du modèle
- La **fenêtre principale de l'approvisionnement est verrouillée** jusqu'à ce que l'éditeur soit fermé
- Effectuez vos modifications dans la fenêtre de l'éditeur

### Étape 3 : Enregistrer les modifications

![Fenêtre principale verrouillée](/img/template/template_editor_locked_screen.png)

- Enregistrez toutes les modifications dans la fenêtre de l'éditeur
- Fermez l'éditeur pour déverrouiller la fenêtre principale
- Les changements sont répercutés dans l'approvisionnement

---

## Options de téléchargement

### Formats de téléchargement disponibles

- **Modèle brut** – Document Word original sans aucun remplacement
- **Word avec remplacements** – Document Word avec les espaces réservés remplis
- **PDF avec remplacements** – Version PDF du modèle complété

---

## Notes importantes

⚠️ **Comportement de l'éditeur :**
- La page principale de modification de l'approvisionnement est **verrouillée** tant que l'éditeur de modèle est ouvert
- Vous devez fermer la fenêtre contextuelle de l'éditeur pour continuer à travailler sur l'approvisionnement
- Les modifications non enregistrées dans l'éditeur peuvent être perdues

✅ **Meilleures pratiques :**
- Examinez les modèles avant la publication
- Testez le remplacement des espaces réservés
- Utilisez le format de téléchargement approprié à vos besoins
- Gardez les noms de modèles clairs et descriptifs
- Vérifiez l'exhaustivité avant d'utiliser dans les attributions
