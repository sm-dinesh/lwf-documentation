# Créer une procédure simplifiée

Cette section explique comment créer un nouvel appel d'offres de procédure simplifiée dans le système.

---

## 📍 Étape 1 : Accéder au menu Créer

![Menu Créer](/img/simple_procurement/simple_procurement_create_menu.png)

- Ouvrez le module **Procédure simplifiée**  
- Cliquez sur le **Menu Créer** pour commencer un nouvel appel d'offres  

---

## ➕ Étape 2 : Cliquer sur le bouton Créer

![Bouton Créer](/img/simple_procurement/simple_procurement_create_button.png)

- Cliquez sur le bouton **Créer** pour ouvrir le formulaire de création d'appel d'offres  

---

## 📝 Étape 3 : Remplir les détails de l'appel d'offres

![Formulaire de création](/img/simple_procurement/simple_procurement_create_from.png)

Remplissez soigneusement les champs suivants :

### 🔹 Informations de base

1. **Nom**  
   Entrez un nom descriptif pour l'approvisionnement

2. **Note de demande**  
   Sélectionnez la note de demande associée  

3. **Catégorie**  
   Choisissez la catégorie d'approvisionnement appropriée  

4. **Code**  
   Fournissez un code d'appel d'offres unique pour le suivi  

---

### 📄 Contenu et configuration

5. **Termes et conditions**  
   Ajoutez tous les termes et conditions applicables

6. **Critères d'évaluation**  
   Sélectionnez les critères d'évaluation des devis

7. **Modèle**  
   Choisissez un modèle de document si nécessaire  

---

### 📅 Dates importantes

8. **Date de publication**  
   9. **Date de soumission**  
10. **Date d'ouverture**  

**Important :** Assurez-vous que les dates sont logiques :  
- Date de soumission > Date de publication  
- Date d'ouverture ≥ Date de soumission  

---

### 📎 Pièces jointes

11. **Fichiers de l'appel d'offres**  
Téléchargez tous les documents et spécifications requis

---

## 💾 Étape 4 : Enregistrer l'appel d'offres

12. **Enregistrer comme brouillon**  
Cliquez sur **Brouillon** pour enregistrer sans publier  

13. **Publier l'appel d'offres**  
Cliquez sur **Enregistrer** pour publier l'appel d'offres  

---

## Prochaines étapes

Après avoir créé votre approvisionnement :

1. **[Modifier les détails](./edit)** – Apporter des modifications si nécessaire
2. **[Ajouter des invitations de fournisseurs](./simple-procurement-invitations)** – Inviter des fournisseurs à participer
3. **[Publier l'approvisionnement](./publish)** – Rendre l'approvisionnement actif et visible
4. **[Gérer les modèles](./template)** – Configurer les modèles de documents
5. **[Définir les critères d'évaluation](./update_evaluaton_criteria)** – Définir les paramètres d'évaluation
