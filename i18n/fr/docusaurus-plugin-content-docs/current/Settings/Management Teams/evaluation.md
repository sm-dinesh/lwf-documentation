# Évaluation

## Accéder à l'évaluation

- Cliquez sur **Évaluation**

![Évaluation](/img/settings/management_teams/evaluation/management_teams_evaluation.png)

## Liste des membres

![Liste d'évaluation](/img/settings/management_teams/evaluation/management_teams_listing.png)

- Affiche tous les membres du groupe d'évaluation

## Gérer les membres

![Gérer l'équipe d'évaluation](/img/settings/management_teams/evaluation/management_teams_evaluation_listing.png)

Actions disponibles :

1. **Ajouter un nouveau membre**
2. **Supprimer un membre existant**

### Ajouter un membre à l'équipe

- Cliquez sur le bouton **Ajouter (1)**

![Ajouter un membre d'évaluation](/img/settings/management_teams/evaluation/management_teams_evaluation_add_team.png)

- Une fenêtre modale apparaît
- Sélectionnez les utilisateurs dans la liste déroulante
- Après un ajout réussi, la fenêtre modale se ferme automatiquement

## Remarques

- La structure de l'équipe a un impact direct sur les flux d'évaluation et d'approbation
- Les utilisateurs peuvent être mis à jour ou supprimés selon les besoins
