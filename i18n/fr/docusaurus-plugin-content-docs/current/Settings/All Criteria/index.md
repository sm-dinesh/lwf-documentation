# Tous les critères

Les critères sont utilisés pour évaluer les offres d'appel d'offres en fonction de différents paramètres. Ces points d'évaluation aident à évaluer et à comparer les offres efficacement pendant le processus d'appel d'offres.

## Accéder aux critères

![Tableau de bord des paramètres](/img/settings/criteria/settings_dashboard.png)

- Naviguez vers **Paramètres**
- Cliquez sur **Tous les critères**

![Tous les critères](/img/settings/criteria/settings_all_criteria.png)
