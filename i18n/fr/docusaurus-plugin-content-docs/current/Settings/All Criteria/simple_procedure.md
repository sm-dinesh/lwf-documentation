# Critères de passation de marché simple

Ce type contient des champs minimaux :

![Passation de marché simple](/img/settings/criteria/settings_simple_procurement_create.png)

1. **Libellé** – Titre du critère
2. **Description** – Optionnel

### Actions

3. **Annuler** – Ignorer les modifications
4. **Créer un critère** – Enregistrer le critère
