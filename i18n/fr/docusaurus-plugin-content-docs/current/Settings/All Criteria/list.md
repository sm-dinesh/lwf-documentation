# Liste des critères

![Liste des critères](/img/settings/criteria/settings_all_criteria_listing.png)

- Affiche une liste de tous les critères disponibles
- Organisé en fonction des types d'appels d'offres

## Onglets

### Appel d'offres ouvert (par défaut)

![Appel d'offres ouvert](/img/settings/criteria/settings_open_tender.png)

- Onglet actif par défaut
- Affiche les critères liés à l'appel d'offres ouvert

### Autres onglets

- **Appel d'offres négocié**
- **Passation de marché simple**

Les utilisateurs peuvent passer d'un onglet à l'autre pour créer ou gérer des critères pour différents types d'appels d'offres.
