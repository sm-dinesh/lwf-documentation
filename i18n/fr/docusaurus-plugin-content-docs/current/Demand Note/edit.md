# Modifier la Note de demande

La page **Modifier la Note de demande** permet aux utilisateurs de modifier une note de demande existante. Les utilisateurs peuvent mettre à jour les informations saisies précédemment, y compris les détails du budget et d'autres sections créées lors de la configuration initiale.

## Accéder à la page de modification

- Accédez à la page de liste des Notes de demande
- Sélectionnez une Note de demande
- Cliquez sur le bouton **Modifier** depuis la page des détails

## Aperçu

![Modifier la Note de demande](/img/demand_note/demand_note_edit.png)

Cette page fournit des champs modifiables pour mettre à jour la note de demande. Toutes les données saisies précédemment sont pré-remplies pour plus de commodité.

## Sections modifiables

### Détails du budget
- Les utilisateurs peuvent consulter les informations relatives au budget associées à la note de demande.

### Autres informations
- Tous les champs saisis lors de la création de la note de demande peuvent être modifiés au besoin.

## Enregistrer les modifications

- Après avoir effectué les modifications, cliquez sur le bouton **Enregistrer la section**.
- Cela mettra à jour la section correspondante de la note de demande.

## Actions disponibles

En haut de la page, les boutons d'action suivants sont disponibles :

### Retourner
- Dirige l'utilisateur vers la page précédente.

### Voir
- Redirige vers la page des détails de la Note de demande.
- Permet aux utilisateurs de consulter la note de demande complète en mode lecture seule.

### Envoyer pour révision
- Soumet la note de demande pour le flux de travail d'approbation.
- La note passera par différentes étapes d'autorisation telles que :
  - Réviseur
  - Vérificateur
  - Approbateur

### Ajouter un fichier
- Permet aux utilisateurs de télécharger et de joindre des fichiers supplémentaires à la note de demande.

## Remarques

- Assurez-vous que tous les champs obligatoires sont mis à jour avant d'enregistrer.
- Une fois envoyée pour révision, d'autres modifications peuvent être restreintes selon les autorisations du flux de travail.
