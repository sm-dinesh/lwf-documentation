# Voir la Note de demande

La page **Détails de la Note de demande** permet aux utilisateurs de visualiser toutes les informations saisies lors de la création d'une note de demande. Cette page offre un aperçu complet ainsi que des options pour naviguer, modifier ou dupliquer l'enregistrement.

## Accéder à la page des détails

- Accédez à la page de liste des Notes de demande
- Cliquez sur une Note de demande spécifique pour ouvrir sa vue détaillée

## Aperçu

![Détails de la Note de demande](/img/demand_note/demand_note_details.png)

La page des détails affiche toutes les informations organisées en plusieurs onglets pour une meilleure lisibilité et gestion.

## Onglets disponibles

### 1. Informations de base
- Affiche les détails généraux de la note de demande tels que le Numéro de réquisition, l'Émetteur, Émis le, le Statut, le Commentaire, l'Emplacement et la Date prévue.

### 2. Budget
- Affiche les informations relatives au budget associées à la note de demande.

### 3. Articles
- Répertorie tous les articles inclus dans la note de demande avec leurs détails respectifs.

### 4. Fichiers
- Affiche les documents téléchargés ou les pièces jointes liés à la note de demande.

### 5. Activité
- Assure le suivi de toutes les actions et mises à jour effectuées sur la note de demande.

## Actions disponibles

En haut de la page, les boutons d'action suivants sont disponibles :

### Retourner
- Dirige l'utilisateur vers la page précédente ou la vue de la liste.

### Modifier
- Permet de modifier la note de demande existante.
- Ouvre la note de demande en mode édition.

### Dupliquer
- Crée une copie de la note de demande actuelle.
- Utile pour générer rapidement une nouvelle note de demande avec des données similaires.

## Remarques

- Tous les onglets sont en lecture seule sauf si l'option **Modifier** est sélectionnée.
- Assurez-vous de disposer des autorisations appropriées pour modifier ou dupliquer une note de demande.
