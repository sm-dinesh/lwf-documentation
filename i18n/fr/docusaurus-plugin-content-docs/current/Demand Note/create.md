### Création de la Note de demande

-   **Visuel :** 


![Créer une Note de demande](/img/demand_note/demand_note_create.png)

Le formulaire de Note de demande se compose de 8 champs répartis en 3 sections :

#### i. Section Budget
1.  **Sélectionner le budget** – Sélectionnez le budget associé parmi les options disponibles.


#### ii. Informations de base
1.  **Commentaire** – Entrez un commentaire pour la note de demande.
2.  **Référence** – Entrez un code de référence unique qui représente la note de demande.
3.  **Sélectionner la devise** – Sélectionnez la devise valide pour la note de demande.
4.  **Emplacement** – Entrez un emplacement valide.
5.  **Date prévue** – Entrez une date prévue future.

#### iii. Articles d'approvisionnement
Dans cette section, vous devez saisir les détails des articles requis dans la note de demande, ce qui inclut l'Article et la Quantité. Une note de demande peut contenir un seul article ou plusieurs articles.

**Section Enregistrer**

Enregistrez et publiez la note de demande.
