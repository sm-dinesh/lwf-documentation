# Gestion des modèles

## Aperçu

La fonctionnalité **Gestion des modèles** permet aux utilisateurs de gérer, modifier et télécharger des modèles de documents associés à un appel d'offres négocié.  

Les utilisateurs peuvent :
- Ajouter de nouveaux modèles à un appel d'offres
- Modifier les détails des modèles existants
- Afficher les espaces réservés des modèles
- Générer des documents avec ou sans remplacement des espaces réservés
- Télécharger des documents dans plusieurs formats

---

## Accès à la gestion des modèles

1. Ouvrez la **Page d'édition** de l'appel d'offres négocié  
2. Cliquez sur le **Bouton Modèle** pour accéder à la liste des modèles

![Edit Page Template Button](/img/template/template_edit_button.png)

---

## Page de la liste des modèles

![Template List Page](/img/template/template_list_page.png)

La page de la liste des modèles affiche :

- Tous les modèles liés à l'appel d'offres  
- Les noms et descriptions des modèles
- Les actions disponibles pour chaque modèle
- Une option pour ajouter de nouveaux modèles

---

## Ajouter un modèle

### Étape 1 : Cliquez sur le bouton Ajouter un modèle

![Add Template Button](/img/template/template_add_button.png)

- Cliquez sur le bouton **Ajouter un modèle** pour ouvrir la fenêtre contextuelle de sélection

### Étape 2 : Sélectionnez un modèle et ajoutez-le

![Add Template Modal](/img/template/template_add_modal.png)

À l'intérieur de la fenêtre contextuelle :

1. **Menu déroulant de sélection** – Choisissez un modèle dans la liste disponible  
2. **Bouton Ajouter** – Cliquez pour lier le modèle sélectionné à l'appel d'offres  

---

## Actions sur les modèles

![Template Action Buttons](/img/template/template_action_buttons.png)

Pour chaque modèle, les actions suivantes sont disponibles :

1. **Bouton d'action Modifier** – Ouvre les options de modification du modèle  
2. **Bouton Modifier** – Modifier les détails du modèle  
3. **Bouton Espace réservé** – Afficher les espaces réservés utilisés dans le modèle  
4. **Télécharger le modèle** – Télécharger le modèle brut sans remplacement des espaces réservés  
5. **Télécharger Word** – Télécharger le document Word avec les espaces réservés remplacés  
6. **Télécharger PDF** – Télécharger le document PDF avec les espaces réservés remplacés  

---

## Modifier un modèle

### Étape 1 : Cliquez sur l'action Modifier

![Edit Template Modal](/img/template/template_edit_modal.png)

À l'intérieur de la fenêtre contextuelle de modification :

1. **Modification du nom du modèle** – Modifiez le titre du modèle  
2. **Bouton Ouvrir l'éditeur** – Lancez la fenêtre de l'éditeur de modèle  

### Étape 2 : Utiliser l'éditeur de modèle

![Template Editor Window](/img/template/template_editor_window.png)

- Une nouvelle fenêtre contextuelle s'ouvre pour modifier le contenu du modèle
- La **fenêtre principale de l'appel d'offres est verrouillée** jusqu'à ce que l'éditeur soit fermé
- Effectuez vos modifications dans la fenêtre de l'éditeur

### Étape 3 : Enregistrer les modifications

![Locked Main Window](/img/template/template_editor_locked_screen.png)

- Enregistrez toutes les modifications dans la fenêtre de l'éditeur
- Fermez l'éditeur pour déverrouiller la fenêtre principale
- Les modifications sont reflétées dans l'appel d'offres

---

## Options de téléchargement

### Formats de téléchargement disponibles

- **Modèle brut** – Document Word original sans aucun remplacement
- **Word avec remplacements** – Document Word avec les espaces réservés remplis
- **PDF avec remplacements** – Version PDF du modèle complété

---

## Notes importantes

⚠️ **Comportement de l'éditeur :**
- La page principale de modification de l'appel d'offres est **verrouillée** pendant que l'éditeur de modèle est ouvert
- Vous devez fermer la fenêtre contextuelle de l'éditeur pour continuer à travailler sur l'appel d'offres
- Les modifications non enregistrées dans l'éditeur peuvent être perdues

✅ **Bonnes pratiques :**
- Examinez les modèles avant d'entrer dans la phase de négociation
- Testez les remplacements d'espaces réservés
- Utilisez le format de téléchargement approprié à vos besoins
- Gardez les noms de modèles clairs et descriptifs
- Vérifiez la complétude avant de l'utiliser dans les contrats finaux
