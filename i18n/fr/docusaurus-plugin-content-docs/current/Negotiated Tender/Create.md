# Créer un appel d'offres négocié

Cette page guide les utilisateurs à travers le **processus de création d'appel d'offres négocié** dans le système Lwf eTender. Le formulaire est conçu pour collecter toutes les informations nécessaires de manière efficace.

---

## 📍 Étape 1 : Accéder au tableau de bord

![Open Tender Menu](/img/negotiated_tender/neg_tender_dashboard.png)

- Accédez aux **Procédures d'appel d'offres négocié** depuis le tableau de bord principal
- Accédez à la liste des appels d'offres négociés ou créez-en un nouveau


## 📍 Étape 1 : Accéder au menu de création

- Ouvrez le module **Appel d'offres négocié** depuis le tableau de bord principal

![Open Tender Menu](/img/negotiated_tender/neg_tender_list.png)

- Localisez l'option **Créer** pour commencer un nouvel appel d'offres

---

## ➕ Étape 2 : Cliquez sur le bouton Créer

- Cliquez sur le bouton **Créer** pour ouvrir le formulaire de création d'appel d'offres
- Le formulaire vous guidera pour saisir toutes les informations requises

![Open Tender Menu](/img/negotiated_tender/neg_tender_create.png)

---

## 📝 Étape 3 : Remplir les détails de l'appel d'offres

Remplissez les champs suivants avec soin :

### 🔹 Informations de base

1. **Nom**  
   Saisissez un nom d'appel d'offres descriptif qui identifie clairement l'achat

2. **Note de demande**  
   Sélectionnez la note de demande associée qui a déclenché cet appel d'offres

3. **Code**  
   Fournissez un code d'appel d'offres unique pour le suivi et l'identification

4. **Catégorie**  
   Choisissez la catégorie d'achat appropriée (ex. Biens, Services, Travaux)

---

### 📄 Contenu et configuration

5. **Termes et conditions**  
   Définissez les règles, les exigences et les obligations des fournisseurs

6. **Description**  
   Fournissez une description détaillée du périmètre d'achat et des exigences

---

### 📅 Dates importantes

7. **Date de publication**  
   Définissez la date à laquelle l'appel d'offres sera officiellement publié

8. **Date de soumission**  
    Spécifiez la date limite pour que les fournisseurs soumettent leurs propositions

9. **Date d'ouverture**  
    Définissez la date à laquelle les propositions soumises seront ouvertes et examinées

**Important :** Assurez-vous que les dates suivent une séquence logique :
- Date de soumission > Date de publication
- Date d'ouverture ≥ Date de soumission

---

### 📋 Informations complémentaires

10. **Publié dans**  
    Indiquez les plateformes où l'appel d'offres apparaîtra

11. **Publié sur**  
    Indiquez les canaux de publication où l'appel d'offres apparaîtra
---

### 📎 Pièces jointes

12. **Documents de certificat minimum**
    Spécifiez les documents que les fournisseurs doivent télécharger (si requis par le processus)

### 📎 Informations complémentaires

12. **Critères**
   Ajoutez les critères sur la base desquels l'évaluation sera effectuée par les évaluateurs


13. **Modèles**  
    Téléchargez tous les documents d'appel d'offres et les spécifications auxquels les fournisseurs se référeront

---

## 💾 Étape 4 : Enregistrer l'appel d'offres

15. **Enregistrer comme brouillon**  
    Cliquez sur **Brouillon** pour enregistrer sans publier (permet des modifications ultérieures)

16. **Publier l'appel d'offres**  
    Cliquez sur **Enregistrer** pour publier l'appel d'offres et le rendre actif

---

## Prochaines étapes

Après avoir créé votre appel d'offres négocié, vous pouvez :

- **[Modifier l'appel d'offres](./Edit)** – Apportez des modifications si nécessaire avant ou après la publication
- **[Gérer les modèles](./template)** – Ajoutez ou modifiez des modèles de contrat
- **[Configurer les critères d'évaluation](./evaluation_criteria)** – Définissez les paramètres d'évaluation des offres
